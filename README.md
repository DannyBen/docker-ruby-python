# Docker Alpine with Ruby and Python 2+3

[Image  on Docker Hub][1] | [Source on GitHub][2]

## Usage

```shell
# Bash into the container and mount the current directory
$ docker run --rm -it -v$PWD:/app dannyben/ruby-python bash
```

## Available Binaries

- ruby
- python
- python3
- pip
- pip3
